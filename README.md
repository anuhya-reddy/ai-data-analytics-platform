# ai-data-analytics-platform
Full-stack AI analytics platform that translates natural language queries into pandas operations or SQL, enabling interactive data exploration with visualizations and insights.

Core Capabilities:
Analyze sales performance across products, regions, and time
Generate visual insights through charts and summaries
Identify trends and anomalies in sales data
Provide AI-driven explanations and recommendations
Enable natural language-style querying of data

Tech Stack:
Backend: Python, Flask
Data Processing: Pandas, NumPy
Visualization: Matplotlib / Plotly
AI Integration: OpenAI API
Frontend: HTML, CSS, JavaScript
